  What you are looking at. This app was deployed with an encrypted volume: the ciphertext travelled through
  the operator's infrastructure, and the key never did - an authorized wallet sealed the volume key to this
  enclave's attested per-boot identity. By the time this code ran, /enc/<name>
  was an ordinary read-only directory. The app holds no crypto, no keys, no HTTP client - just std::fs.
